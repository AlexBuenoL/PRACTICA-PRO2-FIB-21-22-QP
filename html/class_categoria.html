Descripción detallada

Representa una categoria del circuito. La categoria se puede identificar con un string (name) o con un entero entre 1 y c (id), donde c es el numero de categorias del circuito. También contiene un map con un entero como clave (nivel) y un entero que representa los puntos pertenecientes al nivel.
